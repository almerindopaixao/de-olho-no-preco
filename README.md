# De Olho No Preço 🔍💰

Projeto para monitoramento, comparação de preços e envio de ofertas automatizado.

## 🛠️ Tecnologias
- **Python**: Processamento.
- **n8n**: Orquestrador de fluxo.
- **MongoDB**: Armazenamento
- **Docker**: Orquestrador de ambiente

## 🚀 Como Rodar
1. Configure o arquivo `.env`.
2. Execute o comando: `docker-compose up -d --build`.
